# CursoPro 高级版

CursoPro 高级版是一款面向 Cursor 用户的本地辅助工具，主打简洁启动、额度展示和高级模型使用体验。打开应用后，一键开启服务即可配合 Cursor 使用，无需在复杂页面之间来回切换。

## 主要特点

- 支持 Cursor 高级模型使用
- 一键开启 / 关闭本地服务
- 首页显示额度使用情况，界面简洁清爽
- 支持自动检测更新
- 支持 macOS 和 Windows
- 保留使用教程入口，方便快速查看配置方式

## 使用方式

1. 下载对应系统的安装包并解压。
2. 启动 CursoPro 高级版。
3. 点击首页的“开启服务”。
4. 打开 Cursor，即可开始使用。
